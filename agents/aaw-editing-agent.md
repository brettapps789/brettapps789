# ✏️ AAW Editing Agent

**Repo Name:** `aaw-editing-agent`  
**Role:** Quality Refinement

## Primary Capabilities

- Grammar, flow, and readability enhancement
- Content expansion/contraction
- Audience-specific structural optimization

## Description

The Editing Agent refines raw drafts into publication-ready content. It corrects grammar, improves sentence flow, adjusts reading level, and restructures content for the specific audience — whether that's a beginner consumer guide or an advanced technical manual.

## Inputs

- Raw draft from Writing Agent
- Target audience reading level
- Word count constraints

## Outputs

- Polished, publication-ready manuscript
- Readability scores (Flesch-Kincaid, etc.)
- Structural change summary

## Integration Points

- Receives data from: **Writing Agent**
- Feeds data to: **Fact-Checking Agent**, **Formatting Agent**
