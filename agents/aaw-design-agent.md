# 🎨 AAW Design Agent

**Repo Name:** `aaw-design-agent`  
**Role:** Visual Identity

## Primary Capabilities

- Cover art prompts & SVG code generation
- Layout, color palette, & font selection
- Text-to-image visual descriptions

## Description

The Design Agent crafts the visual identity of every product. It generates detailed cover art prompts for text-to-image tools, produces raw SVG code for scalable graphics, and defines complete design systems — color palettes, typography, and layout grids — that maintain brand consistency across all deliverables.

## Inputs

- Market analysis output (title, audience, niche)
- Brand colors or style preferences
- Platform specifications (KDP, web, social)

## Outputs

- Cover art generation prompts
- SVG code for logos/icons
- Design system (colors, fonts, layout)

## Integration Points

- Receives data from: **Market Analysis Agent**
- Feeds data to: **Formatting Agent**, **Web Dev Agent**
