# 📊 AAW Market Analysis Agent

**Repo Name:** `aaw-market-analysis-agent`  
**Role:** Positioning & Strategy

## Primary Capabilities

- Niche profitability scoring
- Title, subtitle, & cover concept generation
- Audience segmentation & pricing tiers

## Description

The Market Analysis Agent transforms raw research data into actionable positioning strategy. It scores niche opportunities, recommends optimized titles and cover concepts, and defines clear audience segments with appropriate pricing models.

## Inputs

- Research Agent keyword and pain point data
- Competitor analysis reports
- Target platform (Amazon KDP, Gumroad, Stripe, etc.)

## Outputs

- Niche profitability score (0–100)
- Title/subtitle recommendations
- Audience segment profiles
- Pricing tier strategy

## Integration Points

- Receives data from: **Research Agent**
- Feeds data to: **Writing Agent**, **Design Agent**, **Integration Agent**
