# 🔎 AAW SEO Agent

**Repo Name:** `aaw-seo-agent`  
**Role:** Search Visibility

## Primary Capabilities

- Meta tags & Schema.org markup
- H1–H6 structural optimization
- Alt-text & keyword placement analysis

## Description

The SEO Agent maximizes organic search visibility for every piece of content and every landing page. It injects optimized meta tags, generates Schema.org structured data, audits heading hierarchy, and ensures all images carry descriptive alt-text with strategic keyword placement throughout.

## Inputs

- Final HTML/content files from Web Dev Agent and Formatting Agent
- Keyword lists from Research Agent
- Target platform and audience

## Outputs

- SEO-optimized HTML with meta tags
- Schema.org JSON-LD markup
- Heading structure audit report
- Alt-text recommendations

## Integration Points

- Receives data from: **Research Agent**, **Web Dev Agent**, **Writing Agent**
- Feeds data to: **Deployment Agent**, **Testing Agent**, **Analytics Agent**
