# 🌐 AAW Web Dev Agent

**Repo Name:** `aaw-web-dev-agent`  
**Role:** Landing Page Construction

## Primary Capabilities

- HTML/CSS/JS generation (Jekyll/Static)
- Mobile-responsive layout design
- Hero section & CTA construction

## Description

The Web Dev Agent builds the sales and landing page infrastructure for every product. It generates clean, production-ready HTML/CSS/JS code compatible with Jekyll and static site generators, produces fully mobile-responsive layouts, and crafts high-converting hero sections with clear calls-to-action.

## Inputs

- Product title, description, and pricing from Market Analysis Agent
- Design system (colors, fonts) from Design Agent
- Stripe checkout links from Integration Agent

## Outputs

- Complete landing page HTML/CSS/JS files
- Mobile-responsive layout code
- Hero section with CTA buttons

## Integration Points

- Receives data from: **Design Agent**, **Market Analysis Agent**, **Integration Agent**
- Feeds data to: **SEO Agent**, **Deployment Agent**
