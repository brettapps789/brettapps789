# ✍️ AAW Writing Agent

**Repo Name:** `aaw-writing-agent`  
**Role:** Content Generation

## Primary Capabilities

- Drafting chapters, intros, and conclusions
- Natural SEO keyword integration
- Tone & style adaptation

## Description

The Writing Agent is the core content engine of the AAW pipeline. It generates full-length drafts — from chapter outlines to polished prose — while naturally weaving in SEO keywords and adapting tone to match the target audience's expectations.

## Inputs

- Market analysis output (title, audience, niche)
- Keyword list from Research Agent
- Desired tone, style, and word count

## Outputs

- Full manuscript drafts (chapters, intro, conclusion)
- SEO-integrated content sections
- Style-adapted copy variations

## Integration Points

- Receives data from: **Research Agent**, **Market Analysis Agent**
- Feeds data to: **Editing Agent**, **SEO Agent**
