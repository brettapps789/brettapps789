# 🔍 AAW Research Agent

**Repo Name:** `aaw-research-agent`  
**Role:** Data Intelligence

## Primary Capabilities

- Keyword research (Google Trends / Amazon)
- Competitor & forum analysis
- Audience pain point identification

## Description

The Research Agent is the intelligence-gathering foundation of the AI Agent Workforce. It surfaces high-demand topics, validates niche viability, and maps audience needs by scanning keyword data, competitor landscapes, and community discussions.

## Inputs

- Target niche or topic
- Platform preferences (Google, Amazon, Reddit, etc.)
- Audience persona descriptions

## Outputs

- Keyword lists with volume/trend data
- Competitor analysis summaries
- Audience pain point reports

## Integration Points

- Feeds data to: **Market Analysis Agent**, **Writing Agent**, **SEO Agent**
