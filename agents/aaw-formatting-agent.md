# 📦 AAW Formatting Agent

**Repo Name:** `aaw-formatting-agent`  
**Role:** Distribution Prep

## Primary Capabilities

- Exporting to PDF, EPUB, and HTML
- Applying brand headers/footers
- Device-responsive text optimization

## Description

The Formatting Agent prepares the final content for distribution across all target platforms and devices. It converts manuscripts into PDF, EPUB, and HTML formats, applies consistent brand elements (headers, footers, logos), and optimizes typography for readability on mobile, tablet, and desktop.

## Inputs

- Final verified manuscript
- Brand design system from Design Agent
- Target platforms (KDP, Gumroad, web)

## Outputs

- PDF (print-ready and digital)
- EPUB (e-reader compatible)
- HTML (web embeddable)

## Integration Points

- Receives data from: **Editing Agent**, **Fact-Checking Agent**, **Design Agent**
- Feeds data to: **Deployment Agent**, **Web Dev Agent**
